# 💸 Salary Splitter (Expense Splitter)

A modern and colorful Salary/Expense Splitter built using HTML, CSS, and JavaScript. This app helps users split expenses easily among multiple people with tip calculation, rounding options, and local history storage.

---

## 🚀 Features

- 💰 Split expenses among multiple people  
- ➕ Adjustable number of people  
- 💡 Tip calculation (preset + custom)  
- 🔄 Round per person option  
- 📊 Real-time calculation updates  
- 💾 Save history using localStorage  
- 📱 Clean, responsive UI design  
- ⚡ Fast and lightweight (no external libraries)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Browser LocalStorage

## 🎯 Usage

1. Enter total amount  
2. Select number of people  
3. Choose or customize tip percentage  
4. Enable rounding if needed  
5. View:
   - Tip amount  
   - Total with tip  
   - Per person split  

6. Save split to history (stored locally)
